# PlantCLEF2026

This is our repo for the kaggle competition PlantCLEF2026, which is about classifying photos of plants. 
This `readme` is a work in progress. The competition is [Here](https://www.kaggle.com/competitions/plantclef-2026).

## Organization 
- `/docs`: markdown, latex, and figures for documentation on our ideas and final models
- `/notebooks`: Experimental notebooks for testing ideas, each should have some documentation cells, and be well nammed. 
- `scripts`: Programing scripts used for training, augmenting, the models themselves. 
- `saved_models`: Saved weights, for some of our models.
